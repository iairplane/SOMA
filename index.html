SOMA: Strategic Orchestration and Memory-Augmented System for Vision-Language-Action Model Robustness via In-Context Adaptation

📄 Paper (arXiv) ⭐ GitHub Code 📜 MIT License

SOMA upgrades frozen Vision-Language-Action (VLA) policies for robust in-context adaptation without parameter fine-tuning, via contrastive Dual-Memory RAG, an Attribution-Driven LLM Orchestrator, Dynamic MCP Interventions, and Offline Memory Consolidation.

🚀 Core Highlights

🧠

Dual-Memory RAG

Contrastive retrieval from both success and failure memory banks for stable, low-cost intervention planning.

🎯

Attribution-Driven Orchestrator

LLM-based strategic planner that generates precise tool-chain plans and task refinements.

🛠️

Dynamic MCP Interventions

Extensible visual tools (overlay, distractor removal) and control flow (rollback, decomposition).

🎥 SOMA Framework

Here, we demonstrate how SOMA perceives visual input and retrieves historical experience to handle image/task prompts without requiring VLA fine-tuning, leading to a higher success rate.

Figure 1: Overview of the SOMA framework architecture.

📊 Generalization Performance Demos

Our experiments are based on the LIBERO Benchmark. We modified LIBERO to create LIBERO-SOMA and also utilized LIBERO-PRO to evaluate robustness.
